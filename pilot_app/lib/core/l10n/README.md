# l10n (localização)

- **Padrão:** pt-BR.
- **Estrutura para en:** `AppStrings` em `app_strings.dart`; `_StringsEn` para inglês. Novas chaves: adicionar em `AppStrings` e em `_StringsPt` / `_StringsEn`.
- Para migrar para ARB (gen-l10n): criar `lib/l10n/app_pt.arb` e `app_en.arb` e configurar `l10n.yaml`.
