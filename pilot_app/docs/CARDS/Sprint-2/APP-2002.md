# APP-2002: Route Planning — UI (Nova rota, paradas, constraints)

## História

Como **usuário**, quero informar origem, destino e paradas em uma tela clara, com opção de reordenar e definir restrições (pedágio, túneis, etc.).

## Critério de aceite

- Tela "Nova rota" funcional; usuário consegue adicionar até 1000 paradas, reordenar, definir constraints e disparar "Calcular rota".

## Subtasks

- [ ] Tela "Nova rota": seleção de origem e destino (busca por endereço ou mapa)
- [ ] Adicionar paradas (até 1000), reordenar por arrastar; cada parada com endereço/coordenada
- [ ] Constraints opcionais: evitar pedágio, túneis, duração máxima, distância máxima
- [ ] Data/hora de partida (opcional); validação de >= agora
- [ ] Botão "Calcular rota": validar no client, chamar API, navegar para tela de resultado ou mapa
- [ ] Loading e tratamento de erro (ex.: "Máximo de 1000 paradas excedido")
- [ ] (Opcional) Cache da última route_request; histórico de requests para reutilizar origem/destino
