# APP-2002: Route Planning — UI (Nova rota, paradas, constraints)

## História

Como **usuário**, quero informar origem, destino e paradas em uma tela clara, com opção de reordenar e definir restrições (pedágio, túneis, etc.).

## Critério de aceite

- Tela "Nova rota" funcional; usuário consegue adicionar até 1000 paradas, reordenar, definir constraints e disparar "Calcular rota".

## Subtasks

- [x] Tela "Nova rota": origem e destino por coordenadas (lat/lon); link na home
- [x] Adicionar paradas (até 1000), reordenar por arrastar; cada parada com lat/lon
- [x] Constraints opcionais: evitar pedágio, túneis, duração máx. (s), distância máx. (m)
- [x] Data/hora de partida (opcional); date/time picker; validação >= agora no validator
- [x] Botão "Calcular rota": validar no client, chamar API, navegar para /routes/result?id=&status=
- [x] Loading e tratamento de erro (mensagem de validação ou genérica)
- [ ] (Opcional) Cache/histórico para reutilizar origem/destino
