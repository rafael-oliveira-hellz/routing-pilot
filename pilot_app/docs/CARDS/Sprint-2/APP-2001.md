# APP-2001: Route Planning — API e modelos

## História

Como **usuário**, quero criar uma solicitação de rota (origem, destino, paradas) para que o backend otimize e retorne a rota.

## Critério de aceite

- Modelos e validação client-side alinhados ao backend; POST /api/v1/route-requests com sucesso para até 1000 paradas.

## Subtasks

- [x] Modelos: `RouteRequestDto`, `RoutePointDto`, `RouteStopDto`, `RouteConstraintDto`, `RouteRequestResponse` (route_dto.dart)
- [x] Validação no app: exatamente 2 points (1 origem, 1 destino); máximo 1000 stops; departure_at >= now; coordenadas válidas (RouteRequestValidator)
- [x] POST /api/v1/route-requests: body com points, stops, constraints, departure_at; traceId via ApiClient
- [x] Resposta: id e status; 4xx tratado como ValidationException no ApiClient
