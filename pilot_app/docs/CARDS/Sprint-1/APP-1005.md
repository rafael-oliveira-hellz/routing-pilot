# APP-1005: Auth — Logout, Refresh token, Forgot/Reset/Change password

## História

Como **usuário**, quero sair da conta, renovar a sessão automaticamente quando o token expirar, redefinir senha por email e alterar senha quando já estou logado.

## Critério de aceite

- Logout limpa storage e navega para login; interceptor faz refresh em 401; telas de esqueci senha, redefinir e alterar senha funcionais.

## Subtasks

- [x] **Logout:** Botão no menu; POST /api/v1/auth/logout com Bearer; limpar secure storage; navegar para login
- [x] **Refresh token:** Interceptor Dio: em 401 chama tryRefreshAndSave (POST refresh); se 200 salva tokens e reenvia a requisição; se falha → logout (limpar storage) e rejeitar com AuthException
- [x] **Forgot password:** Tela "Esqueci a senha" com email; POST /api/v1/auth/forgot-password; mensagem genérica. Tela "Redefinir senha" (/reset-password?token=): nova senha + confirmação; POST reset-password; sucesso → login com message=reset_ok
- [x] **Change password:** Tela "Alterar senha" (link no AppBar da home): senha atual, nova, confirmar; POST /api/v1/auth/change-password com Bearer; validação client-side; feedback (senha atual incorreta / sucesso)
- [x] Documentar endpoints e fluxos em lib/features/auth/data/README.md
