# APP-1006: Auth — Revogar todas as outras sessões (perda/roubo do celular)

## História

Como **usuário**, quero encerrar o acesso em todos os outros dispositivos a partir deste (ex.: celular perdido ou roubado) e manter apenas este aparelho logado.

## Critério de aceite

- Tela Segurança/Sessões com botão que chama revoke-all-other-sessions; novo refreshToken atualizado no storage; mensagem de sucesso.

## Subtasks

- [ ] Tela de Segurança ou Sessões (ex.: dentro de Perfil ou Configurações) com botão "Encerrar sessões em todos os outros dispositivos"
- [ ] POST /api/v1/auth/revoke-all-other-sessions com Bearer
- [ ] Se a resposta trouxer novo refreshToken, atualizar no secure storage para que **este** aparelho continue logado
- [ ] Exibir mensagem de sucesso (ex.: "As outras sessões foram encerradas. Este aparelho continua logado.")
- [ ] Caso de uso documentado: usuário perdeu ou foi roubado o celular e, a partir de outro dispositivo, encerra o acesso no aparelho perdido/roubado
