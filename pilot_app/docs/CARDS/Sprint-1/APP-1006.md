# APP-1006: Auth — Revogar todas as outras sessões (perda/roubo do celular)

## História

Como **usuário**, quero encerrar o acesso em todos os outros dispositivos a partir deste (ex.: celular perdido ou roubado) e manter apenas este aparelho logado.

## Critério de aceite

- Tela Segurança/Sessões com botão que chama revoke-all-other-sessions; novo refreshToken atualizado no storage; mensagem de sucesso.

## Subtasks

- [x] Tela de Segurança ou Sessões (ex.: dentro de Perfil ou Configurações) com botão "Encerrar sessões em todos os outros dispositivos"
- [x] POST /api/v1/auth/revoke-all-other-sessions com Bearer
- [x] Se a resposta trouxer novo refreshToken (e accessToken), atualizar no secure storage para que **este** aparelho continue logado
- [x] Exibir mensagem de sucesso (ex.: "As outras sessões foram encerradas. Este aparelho continua logado.")
- [x] Caso de uso documentado em lib/features/auth/data/README.md e na tela Segurança
